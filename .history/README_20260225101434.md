# My Wallet Project

This is a fullstack project structured as a monorepo.

## Project Structure

- **client/**: React Native / Expo frontend application.
- **server/**: Backend API (Coming soon).

## Getting Started

To run the frontend:

1. Navigate to the client directory:
   ```bash
   cd client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the project:
   ```bash
   npx expo start
   ```
